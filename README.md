# Gear Tone
### Dia 10/11/2025 - Projeto Pessoal GEARTONE
- Dado inicio a primeira fase do projeto pessoal Gear Tone.

#### Qual a função do Gear Tone?
- O Gear Tone é que um administrador que faz gerenciamentos de equipamentos musicais de um músico.
Possibilitando o mesmo ter uma lista de controle de todos os seus equipamentos.
- Também possibilita ao músico salvar os parametros de seus timbres de pedais e pedaleiras.
Baseando-se num tipo de "Google Keep" só que focado em setup e timbres.
