# Gear Tone

### Dia 10/11/2025 - Projeto Pessoal GEARTONE

- Dado inicio a primeira fase do projeto pessoal Gear Tone.

#### Qual a função do Gear Tone?

- GearTone é um sistema para ajudar músicos a organizar seus estudos, repertório e equipamentos, evitando que informações importantes fiquem espalhadas em cadernos, notas do celular ou na memória.
